# Weather App

Una aplicación del tiempo desarrollada con Ionic y Angular, que permite consultar el pronóstico actual y por horas, con soporte para múltiples idiomas (Inglés y Español).

## Diseño Original

El diseño inicial de la aplicación se basa en este prototipo de Figma:
[Ver diseño en Figma](https://www.figma.com/design/jFRsK1ZCQwQJHJJin0o7j8/Google-Material-Design--Community-?node-id=213-2&m=dev&t=roLR2CEXRQaDUp0h-1)

## Características

- **Pronóstico Actual**: Temperatura, descripción del clima e icono.
- **Pronóstico por Horas**: Detalle de la evolución del clima en las próximas horas.
- **Predicción 5 Días**: Lista con las temperaturas máximas y mínimas.
- **Multi-idioma**: Soporte completo para Español (ES) e Inglés (EN).
- **Geolocalización**: Detección automática de la ubicación del usuario.
- **Búsqueda**: Posibilidad de buscar el tiempo en cualquier ciudad.

## Tecnologías

- Ionic Framework 8
- Angular 19
- OpenWeatherMap API
- ngx-translate

## Instalación

1. Clonar el repositorio.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar el servidor de desarrollo:
   ```bash
   ionic serve
   ```
